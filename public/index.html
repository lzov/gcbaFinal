📘 API RESTful de Productos

Bienvenido a la documentación inicial de esta API. Abajo encontrarás rutas útiles para probarla.

Base URL: https://gcbafinal-ecommerce.onrender.com

🔗 Endpoints públicos

  • GET /api/products — Listar todos los productos
  • GET /api/products?nombre=Reloj — Buscar por nombre
  • GET /api/products?categoria=Deportivo — Filtrar por categoría
  • GET /api/products?minPrecio=5000 — Precio mínimo
  • GET /api/products?maxPrecio=20000 — Precio máximo
  • GET /api/products?orden=asc — Ordenar por precio ascendente
  • GET /api/products?orden=desc — Ordenar por precio descendente

🔐 Autenticación

  • POST /api/auth/login
  • POST /api/auth/register
  • GET /api/auth/profile (requiere token)

⚠️ Reglas importantes

  • No usar ?q junto con filtros de precio.
  • Se priorizan los filtros de precio sobre otros parámetros.
